+ Why are you making this website?

I saw Rose's website, and decided to one up her word to html. I previously
made a host page for an IoT project I was doing, so I had a little experience
in HTML and CSS. I decided to learn javascript as well for fun, originally
to make distributable html tools but right now I'm making games.

+ Did you make the stuff here?

Almost all of the HTML and CSS were from scratch, like opening a text
editor and start typing scratch. I used the icons8 line-awesome library for
for the side menu icons and created others in photoshop. I wrote the executables
in C#, I combined the PDF notes using my PDFCombine tool and OCRd them with a
standard PDF viewer. I wrote all the games in pure javascript using canvases
This accordion you are currently interacting with is from jQuery and the syntax
highlighting on the code section is done using prismjs.
